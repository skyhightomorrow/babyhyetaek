## 데이터 소스 (03 검증 완료 2026-07-14)

### A. 지자체 지원금 — 한국사회보장정보원 지자체복지서비스 API (검증됨)
- Base: `http://apis.data.go.kr/B554287/LocalGovernmentWelfareInformations/`
- 목록 `LcgvWelfarelist?lifeArray=001|002|007` — 001 영유아 856건 · 002 아동 831건 · 007 임신·출산 439건 (2026-07-14 실측, lastModYmd 당일 = 살아있는 데이터)
- 상세 `LcgvWelfaredetailed?servId=WLF...` — **alwServCn(급여내용·금액!)**, sprtTrgtCn(대상),
  slctCritCn(선정기준), aplyMtdCn(신청방법·기한), baslawList(근거 조례), enfcBgngYmd(시행일),
  inqplCtadrList(담당부서·전화), lastModYmd
- 키: 임시로 buddy 프로젝트 DATA_GO_KR_KEY 재사용 (일 1,000건 한도) → **F 전용 키 발급 필요**(자동승인, 사용자)
- 상세 ~1,700건 수집 = 한도 때문에 2일 분할 or 전용 키
- ⚠️ 아이사랑 게시판(childcare.go.kr bbsCd=9091, 227건)은 상세보기 엔드포인트가 curl·자체 폼 제출
  모두 차단/고장(2026-07-14 확인) — API로 대체. 게시판은 교차검증용 참고만.
